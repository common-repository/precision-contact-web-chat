1. Plugin controls appear in the Settings section.

3. The plugin creates this "Start a chat!" button on your home page.
4. When a website visitor clicks the Start button, it opens the chat box and lets them start a conversation with you.


== Description ==

Text-enable your business phone number(s). Webchat-enable your website. Receive customer texts and chat messages on your mobile phone or browser.

To subscribe to the Precision Contact service, visit https://www.messagemybiz.com/.

For instructions on obtaining your Web Chat key, visit https://pcwebchat.messagemybiz.com/

Terms of Service - https://www.messagemybiz.com/terms.html

== Installation ==

Upload the Precision Contact Web Chat plugin to your site, activate it, and then enter your Web Chat key.
